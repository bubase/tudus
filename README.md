# Тудус! (todos)
Простое приложение для списка дел.

**Стек:** Vue 2.6, Vuex, Vue CLI.

Попробовать: https://bubase.github.io/tudus

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
